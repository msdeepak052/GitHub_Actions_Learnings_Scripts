## ğŸ” Variables vs Secrets: What's the Difference?

| Feature    | Variables (`env`)                   | Secrets (`secrets`)                      |
| ---------- | ----------------------------------- | ---------------------------------------- |
| Used for   | Config values (e.g., region, env)   | Sensitive info (e.g., API keys, tokens)  |

| Security   | Not encrypted                       | Encrypted and secure                     |
